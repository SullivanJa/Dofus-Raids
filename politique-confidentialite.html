Politique de confidentialité

Dernière mise à jour : 24 juin 2026

1. Présentation

Cette politique de confidentialité explique quelles données sont collectées et utilisées par Dofus Raids, un bot Discord destiné à organiser des raids Dofus.

Le bot n'est pas affilié, associé, autorisé, approuvé ou sponsorisé par Ankama, Dofus, Discord ou DofusDB.

2. Données collectées

Le bot peut enregistrer certaines données nécessaires à son fonctionnement, notamment :

  • l'identifiant Discord de l'utilisateur
  • le pseudo Discord ou nom affiché
  • l'identifiant du serveur Discord
  • les personnages Dofus ajoutés par l'utilisateur
  • les classes, spécialisations et informations de stuff des personnages
  • les inscriptions aux raids
  • les absences déclarées
  • les personnages sélectionnés dans les rosters
  • l'historique de participation aux raids
  • les rôles Discord configurés comme administrateurs raid
  • le rôle Discord configuré pour les annonces de roster
  • certaines informations techniques nécessaires au bon fonctionnement du bot

3. Données non collectées

Le bot ne collecte pas volontairement :

  • les mots de passe
  • les adresses email des utilisateurs Discord
  • les messages privés
  • les informations de paiement
  • les données sensibles sans rapport avec l'organisation des raids

4. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • permettre l'inscription aux raids
  • gérer les personnages et profils joueurs
  • composer et valider des rosters
  • afficher l'historique de participation
  • gérer les permissions liées aux raids
  • assurer le bon fonctionnement technique du bot
  • prévenir les erreurs, abus ou mauvaises utilisations du service

5. Base de données et hébergement

Les données du bot sont stockées dans une base de données utilisée par le bot. L'accès à cette base est limité aux personnes responsables de l'administration technique du bot.

Le bot peut être hébergé sur un service tiers. Dans ce cas, certaines données techniques peuvent transiter par l'infrastructure de l'hébergeur afin de permettre le fonctionnement du service.

6. Partage des données

Les données ne sont pas vendues. Elles ne sont pas partagées avec des tiers à des fins publicitaires.

Certaines informations peuvent être visibles par les membres du serveur Discord lorsque le bot affiche des inscriptions, absences, personnages proposés, rosters ou historiques de participation.

7. Icônes et API DofusDB

Le bot peut utiliser des icônes provenant de l'API DofusDB, notamment pour représenter des classes ou éléments liés à Dofus. Ces icônes et ressources visuelles appartiennent à leurs ayants droit respectifs.

L'utilisation de ces icônes sert uniquement à améliorer la lisibilité des informations affichées par le bot. Dofus Raids Bot n'est pas affilié à DofusDB.

8. Durée de conservation

Les données sont conservées tant qu'elles sont nécessaires au fonctionnement du bot sur le serveur Discord concerné, ou jusqu'à leur suppression par l'utilisateur ou par un administrateur autorisé.

9. Suppression des données

Les utilisateurs peuvent supprimer leurs données personnelles enregistrées par le bot via la commande :

/profil_supprimer_mes_donnees

Cette commande permet de supprimer les données liées au profil utilisateur, notamment les personnages, inscriptions et informations associées à l'utilisateur.

Pour toute demande supplémentaire concernant les données, vous pouvez contacter l'éditeur du bot sur discord : arkaelid#6173.

10. Sécurité

Des mesures raisonnables sont mises en place pour limiter l'accès aux données et éviter leur perte, leur modification non autorisée ou leur divulgation. Cependant, aucun système informatique ne peut être garanti comme totalement sécurisé.

11. Services tiers

Le bot fonctionne avec Discord et peut utiliser des ressources provenant de services tiers, notamment DofusDB pour certaines icônes. L'utilisation de Discord est également soumise aux conditions et politiques de Discord.

12. Modification de la politique

Cette politique de confidentialité peut être modifiée à tout moment. La version publiée sur cette page est la version applicable.

13. Contact

Pour toute question concernant cette politique de confidentialité ou les données traitées par le bot, vous pouvez contacter l'éditeur du bot sur discord : arkaelid#6173.
